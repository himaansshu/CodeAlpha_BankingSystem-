# 🏦 Bank Management System (C++)

A fully functional **Bank Management System** built using C++ that simulates real-world banking operations through a console-based interface. This project allows users to create accounts, perform secure login, and execute essential banking operations such as deposits, withdrawals, and fund transfers. It uses file handling for persistent data storage and demonstrates practical implementation of object-oriented programming concepts.

---

## 🚀 Features

* 🔐 Secure Login & Account Creation
* 💰 Deposit & Withdrawal Operations
* 🔁 Fund Transfer Between Accounts
* 📜 Transaction History with Timestamp
* 👤 Customer Details Management
* 💾 File Handling for Data Persistence
* 🎨 Colored Console UI for Better Experience

---

## 🛠️ Tech Stack

* **Language:** C++
* **Concepts Used:** OOP (Classes & Objects), File Handling, STL (Vector), Time Functions

---

## 📂 Project Structure

* `Account Class` → Handles account operations (login, deposit, withdraw, transfer)
* `Customer Class` → Stores user details
* `Transaction Class` → Manages transaction records with timestamps
* File Storage → Each account is stored in a separate `.txt` file

---

## ⚙️ How It Works

1. User creates a new account (initial balance ₹500)
2. Unique account number is generated automatically
3. User logs in using account number and password
4. After login, user can:

   * Deposit money
   * Withdraw money
   * Transfer funds
   * View account details
   * Check transaction history

---

## ▶️ How to Run

1. Clone the repository
2. Open the project in any C++ compiler (VS Code / CodeBlocks)
3. Compile the code:

   ```bash
   g++ main.cpp -o bank
   ```
4. Run the program:

   ```bash
   ./bank
   ```
